# 🐾 PuppyStudio

Small experimental studio focused on multiplayer systems, tooling, and game-adjacent infrastructure.

---

## 🚧 What we do

- Build multiplayer experiences using modern engines (currently s&box)
- Explore real-time systems (networking, authority, state)
- Develop internal tools and developer tooling
- Experiment with self-hosted infrastructure and CI/CD pipelines

---

## 🎮 Current Projects

### Breach 2 (s&box)

A prototype asymmetric multiplayer gamemode inspired by containment breach scenarios.

- Humans vs SCP gameplay loop
- Server-authoritative systems
- Systems-first design, not just gameplay

**Status:** Early prototype (MVP)

---

## 🛠️ Tooling & Experiments

We also build small tools to improve developer experience:

- VS Code extensions for s&box workflows *(planned)*
- Internal utilities for faster iteration
- Infrastructure experiments (self-hosting, automation, deployment)
